## 3. 策略标签词典

> 避免每次归因重新造词，统一如下：

| 一级标签 | 二级标签 | 描述 |
|---|---|---|
| 价值 | 深度价值 / 质量价值 / GARP / 红利 | 低估值或高分红偏好 |
| 成长 | 消费 / 科技 / 医药 / 新能源 / 制造 | 押注高增长行业 |
| 均衡 | 全行业选股 / 行业轮动 | 不押注单一赛道 |
| 主题 | 行业主题型 | 高度集中 |
| 小盘 | 微盘 / 中小盘 | 偏好小流通市值 |
| 量化 | 主观+量化辅助 | 有量化模型支撑 |
