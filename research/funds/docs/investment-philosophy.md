## 六条论点（用户原话）

1. **不认为长期有任何一种策略能大幅超额跑赢市场。**
2. **也不认为有任何人能征服市场。**
3. 但可以确定的是：**一定在某个阶段、某个时期，那些主动基金的基金经理——他们的策略、眼光、战术——获取到了超额收益。**
4. **真正的复利，来自波段收益的叠加。**
5. 因此需要不断分析基金，**确认能识别这些基金和这个市场真正在炒的是什么**。
6. 核心信念：**在金融市场上，真正安全的地方，就是钱最多的地方——流动性最充裕、信心最足的地方。**

> 论点 1、2 是**怀疑**（别迷信长期赢家、别迷信任何人）；论点 3 是**确信**（阶段性超额真实存在）——这份确信是整个项目存在的理由；论点 4–6 是**方法论**（怎么把确信变成复利）。

### 论点 #6 释义（关键 · 防误读）

「钱最多 / 流动性最充裕 / 信心最足」描述的是基金**所投资的板块（市场）**，**不是基金自身的规模**。

- 安全 = 把钱放在**高景气度、高流动性**的板块（资金正涌入、深度足、市场信心强的方向），**不是**买规模最大的基金。
- 恰恰相反——**基金自身规模过大（>100 亿）是风险**（侵蚀主动选股灵活性、容量受限），须作为**约束**，不是加分项；过小（<2 亿）有清盘风险。
- 落到分析：「钱最多」= **板块资金流向 / 景气度**（资金堆在哪），「流动性最充裕」= **板块深度 / 大盘属性**，「信心最足」= **市场对该板块的共识信心**——三者都是板块层属性。

## 一句话本质

不押注任何长期赢家，而是**持续识别「当前阶段谁在创造真超额」**，把一个个波段的收益叠加成复利，并始终站在**钱最多、流动性最充裕、信心最足**的那一边。

## 与现有 north-star 的关系（调和，不推翻）

INVARIANTS §b / search-strategy §1 里的「Sharpe 0.40–0.47 / tenure>3y / 每 ~3 年再平衡 / prefer true alpha」**不是与哲学并列的第二套目标，而是本哲学的工程化产物**：

| north-star 条目 | 操作化的是哲学第几条 | 调和 |
|---|---|---|
| 评级 3Y/5Y、tenure>3y、「多年」样本 | #3 | 长周期指标**只用于扩大「可识别波段」的样本**，不断言 α 会延续；最终看当下波段 |
| prefer true alpha（Brinson 选股贡献） | #3、#5 | 真 α ≠「此人能征服市场」，而是「在所考察波段里，**选股**而非**行业 β**贡献了超额」——这正是量化「这基金真正在炒什么」的入口 |
| 每 ~3 年再平衡 | #4 | 再平衡是**组合层**周期参数（backtest 最优）；波段叠加是**单基金层**逐只研判；不同抽象层，并行不悖 |
| 防御层 = 黄金/红利低波 smart-beta | #6 | smart-beta 是**逆周期底仓缓冲**；哲学 #6「跟钱走」是**顺流动性/顺信心的顶层板块择时**——分层而非互斥：识别不到强势流动性主线时，smart-beta 兜底 |

## 否定式边界（本项目不做什么）

- **不追求、也不声称能识别「长期赢家」。** 所有 α 结论都是「**截至当前 asOfDate 的波段判定**」，随 daily diff 与再评估失效。
- 长周期指标只扩大可识别波段的**样本与证据厚度**，不是「相信此人能持续跑赢」的依据。
- 单源晨星 SSR，不含逐日净值/资金流时序——「波段」当前以「年」为最细粒度近似，行情级波段是已知缺口（见第三步设计）。
